# My Admin Dashboard For My Portfolio



This is a private, admin dashboard used to manage all my content displayed on my personal portfolio website.

This dashboard acts as the control center for my portfolio, allowing me to create, update, and delete content securely through a clean and responsive UI.

## Tech Stack 

- React
- Tailwind CSS
- React Router DOM
- TanStack React Query
- Axios
- React Quill (Rich Text Editor)

## Authentication & Security

- Admin-only access
- Protected routes
- Authenticated using **JWT with HTTP-only cookies**
- Directly integrated with the Portfolio API Server

This dashboard is **not accessible to the public** and is intended for admin use only (can only be used by me).

## Features

- Admin authentication
- Protected routing
- Manage portfolio content:
  - Projects (CRUD)
  - Skills (CRUD)
  - Social links (CRUD)
  - About Me content (rich text editing)
- Rich text editing using React Quill
- Optimized API state management with React Query
- Clean, responsive UI built with Tailwind CSS

## Deployment

This project is deployed on Netlify.
